# Page metadata
Simple Metadata will create Page Type options in Page (metabox).

## Simple Metadata settings
Page type have 13 options which allow the configuration of the metadata:
* WebPage *Activated by default*
  * AboutPage
  * CheckoutPage
  * CollectionPage
    * ImageGallery
    * VideoGallery
  * ContactPage
  * FAQPage
  * ItemPage
  * MedicalWebPage
  * ProfilePage
  * QAPage
  * SearchResultsPage
